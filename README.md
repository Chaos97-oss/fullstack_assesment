# VitalSwap Frontend Assessment

## Tech Stack
- React
- Node.js (Express)
- Axios
- CSS (custom, no UI library)

## Features
- Settlement accounts UI replicated from provided design
- Backend API serving settlement account data
- Frontend consumes backend API
- Responsive dashboard layout

## Live Demo
Frontend: https://vitalswapassesment.vercel.app
Backend API: https://fullstack-assesment-qtbf.onrender.com/api/settlement-accounts

## How to Run Locally

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start
